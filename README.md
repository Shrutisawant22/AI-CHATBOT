# 🤖 AI Chat Assistant

A modern, responsive AI chat application built using **HTML, CSS, and Vanilla JavaScript**, powered by the **Google Gemini API**. This application allows users to interact with an AI in real-time through a clean and intuitive chat interface.



## 🚀 Features

- 💬 Real-time AI conversation  
- 🤖 Integration with Google Gemini API  
- ⚡ Model fallback system for reliability  
- ⏳ Typing indicator with loading state  
- ✨ Streaming response effect (character-by-character)  
- 🧠 Chat history stored using localStorage  
- 🌓 Dark / Light theme toggle  
- 🧹 Clear chat functionality  
- 🎯 Predefined suggestion prompts  
- 📱 Fully responsive design (mobile + desktop)  
- 🛡️ Robust error handling (API errors, network issues, quota limits)  



## 🧠 Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Google Gemini API

## ⚙️ Setup Instructions

1. Clone the repository  
git clone https://github.com/Shrutisawant22/AI-CHATBOT.git  

2. Navigate to the project folder  
cd AI-CHATBOT  

3. Add your Gemini API key in script.js  
const API_KEY = "YOUR_API_KEY_HERE";  

4. Open index.html in your browser  



## 🔐 Security Note

Do not expose your API key publicly. Always replace it with a placeholder before pushing code to GitHub.
